# Twitch Viewers Channel & three analytics Streams

This project is made using React, [Axios](https://github.com/axios/axios)  and [Twitch API](https://dev.twitch.tv/docs). 


# Install the project
Go to the root and type `npm install` and hit Enter

# Run the project
Type `npm start` for running project

# Webbrowser
Application runs at `http://localhost:3000`
